# Coffee Shop Sales Performance Dashboard

## Project Overview
The objective of this project is to analyze sales performance using the provided Coffee Shop Sales dataset and build an interactive dashboard in Tableau. The dashboard presents key performance indicators (KPIs), sales trends, category-wise sales, store performance, and highlights business insights to support decision-making.

## Dataset Details
- File Name: Coffee Shop Sales.xlsx  
- Total Records: 149,116  
- Total Columns: 11  
- Time Period: Includes daily transactions  
- Key Columns Used:
  - transaction_date
  - transaction_time
  - transaction_qty
  - unit_price
  - store_location
  - product_category
  - product_type

## Tools Used
- Tableau (for dashboard creation)
- Microsoft Excel (for data review and understanding)

## Dashboard Features
1. KPI Cards:
   - Total Sales
   - Total Transactions
   - Total Quantity Sold
   - Average Sale Per Transaction (calculated KPI)

2. Sales Trend (line chart)
   - Daily and monthly sales variation over time

3. Sales by Day of Week (bar chart)

4. Product Category Analysis (treemap)

5. Store Performance (pie chart)

6. Sales vs Target Analysis (simple bullet chart with reference line)

7. Optional advanced analysis: Box & Whisker chart for sales distribution

## Insights
- Identified best-selling categories and products
- Recognized peak sales days
- Observed store-wise performance contribution
- Evaluated whether target sales were met
- Highlighted sales fluctuation trend over the given period

## Final Outcome
A well-designed, dark-themed and visually clear Tableau dashboard providing complete business insights suitable for decision-making and presentation. The dashboard helps understand performance trends, top-selling products, and store-wise distribution for strategic planning.

## Conclusion
The Coffee Shop Sales Dashboard successfully summarizes key metrics and trends, which can help management improve product planning, store optimization, and set realistic sales targets. The dashboard approach can be extended with advanced interactive filters and forecasting techniques for future enhancement.
